# Survivors Lords

# GAME LOOP

Versión 0.1

---

# Bucle Principal

El jugador comienza sin recursos y debe explorar un mundo hostil.

Cada acción realizada genera progreso tanto para el personaje como para el asentamiento.

---

---

# Micro Loop

Cada combate debe generar una recompensa.

Ejemplos:

• Experiencia

• Materiales

• Oro

• Equipo

• Objetos especiales

---

# Macro Loop

Cada región completada cambia el mundo.

Ejemplos:

Nuevo NPC.

Nuevo edificio.

Nuevo recurso.

Nueva mazmorra.

Nueva historia.

---

# Objetivo del jugador

Al principio:

Sobrevivir.

A mitad del juego:

Expandir el asentamiento.

Al final:

Gobernar un reino completamente reconstruido.
